# Demo GIF / Video Instructions

## Recording Setup
- **Tool:** OBS Studio (free) or Loom (easy sharing)
- **Resolution:** 1920x1080
- **Browser:** Chrome, dark mode
- **Backend:** Running on localhost:8000
- **Frontend:** Running on localhost:3000

## Script (90-120 seconds)

### Scene 1: Landing Page Hero (8s)
1. Open DevPulse landing at localhost:3000/landing
2. Show "Stop API Breaches. Cut AI Costs by 40%." headline
3. Quick scroll through the three pillars

### Scene 2: Security Score Card (15s)
1. Navigate to dashboard
2. Show Security Score Card component
3. Paste code with an exposed OpenAI key: `sk-abc123...`
4. Click "Scan" → show score dropping, token leak detected

### Scene 3: Full Security Scan (15s)
1. Paste code with prompt injection pattern
2. Run full scan → show score, grade, categorized issues
3. Click "Get AI Fix" → show AI-powered fix suggestion

### Scene 4: Threat Feed (8s)
1. Show live threat intelligence feed
2. Expand one threat to show details

### Scene 5: API Inventory (10s)
1. Paste code that imports openai, anthropic, stripe
2. Show auto-detected API providers

### Scene 6: Cost Intelligence Dashboard (15s)
1. Scroll to Cost Intelligence section
2. Show cost breakdown by provider (pie chart)
3. Show cost breakdown by model
4. Show 30-day forecast with sparkline

### Scene 7: ROI Calculator (10s)
1. Set monthly cost slider to $5,000
2. Set team size to 10
3. Show instant ROI calculation: savings, payback period

### Scene 8: VS Code Extension (10s)
1. Switch to VS Code
2. Show inline token leak warning (red squiggly)
3. Open command palette → "DevPulse: Full Security Scan"
4. Show notification with score

### Scene 9: Budget & Pricing (8s)
1. Show Budget Manager with alerts
2. Quick scroll through pricing: Free / Pro $29 / Team $99
